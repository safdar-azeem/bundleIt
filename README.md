### Build from Source

1. Prerequisites:
```bash
# Install Rust
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# Install Node.js (16+) and Yarn
npm install -g yarn
```

2. Clone & Install:
```bash
# Clone repository
git clone https://github.com/safdar-azeem/bundleit.git
cd bundleit

# Install dependencies
yarn install
```

3. Development:
```bash
# Run in development mode
yarn tauri dev

# Build for production
yarn tauri build
```
